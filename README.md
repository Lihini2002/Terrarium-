I'm creating this project as an attempt to see if the theories that the world is a simulation. Here is an artificial life ecosystem simulation
that models evolution in a dynamic environment using a genetic algorithm. 
It features interacting agents such as prey and predators that survive, move, reproduce, and die based on energy constraints and environmental pressures like food availability and predation. 
Each creature is defined by a mutable DNA structure encoding traits such as speed, vision range, turning ability, metabolism, and behavioral tendencies, allowing natural selection to emerge over generations. 
Over time, populations evolve toward more efficient survival strategies as fitness-driven reproduction favors successful genetic traits.
